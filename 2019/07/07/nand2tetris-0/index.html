概述

这是一门基于项目的自下而上构建计算机的有趣的课程。

按照以下层次自下而上讲解每一层的构建，每到一层只关心这一层的实现，不关注下面层次的具体实现。

总体不深入，比较浅显易懂。现代计算机体系结构中的2个重要的概念，流水线和存储层次，几乎没有涉及。
